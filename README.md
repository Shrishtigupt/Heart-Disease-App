# ❤️ Heart Disease Prediction App

A machine learning web application built with Streamlit that predicts the likelihood of heart disease based on user input parameters.

---

## 🚀 Live Demo
https://heart-disease-app-cvgv2hm74azcvhuxioux3v.streamlit.app/

---

## 📌 Overview

This project uses a trained machine learning model to analyze health-related inputs and provide real-time predictions for heart disease risk. It features a simple, clean, and interactive interface.

---

## ✨ Features

- 🔍 Real-time heart disease prediction  
- 🧠 Machine learning-based model  
- 📊 Interactive and user-friendly UI  
- ⚡ Fast and lightweight   

---

## 🧠 Machine Learning Details

- **Algorithm Used:** KNN   
- **Library:** scikit-learn  
- **Model Saving:** joblib  

### Data Processing:
- Handling missing values  
- Encoding categorical variables  
- Feature scaling
